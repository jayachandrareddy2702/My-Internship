My Projects

Here are some of the projects I have worked on:

  • basic login/Register page Website

    A personal and basic project for login/register pages to basically store the User data in the required fields using .Net6.0 framework using C# language. It is executed using visual studio 2022 to deploy this project.

  • Kinetic Clock

    I had done a hardware based project which mainly focuses on clock. The uniqueness of this project is that it displays time using an Arduino UNO, servo motor drivers, clock module, 3D printed seven segment part and servo motors of 28 no.'s. It is majorly made for the people who loves aesthetic styles and uniqueness.

  • Gestures Recognition for smart devices handling

    A major project done in my 3rd year of my Engineering, which mainly an proposed idea for accessing devices remotely without touching the device and it can be operated from a bit of distance from device using Camera to capture the gestures given by the user. It majorly done using Machine learning Algorithms namely:

    • Convolutional Neural Networks (CNN)
    • Support Vector Machine (SVM)
    • Recurrent Neural Networks(RNN)
    • Deep Neural Networks(DNN)
    • K-nearest Neighbor(KNN{pre-existing algorithm})
